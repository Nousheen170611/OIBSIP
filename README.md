Task-2 BMI Calculator

A simple Body Mass Index (BMI) Calculator built using Python that helps users calculate their BMI based on height and weight inputs. It also categorizes the BMI result (Underweight, Normal, Overweight, or Obese) according to standard BMI ranges.

 Features:
Takes user input for height and weight
Calculates BMI using the standard formula
Displays BMI category based on the calculated value
Simple and beginner-friendly Python project
Command-line interface for easy use

 Formula Used:
BMI=Weight(kg)/[Height(m)]^2
Example:
If your weight is 60 kg and height is 1.65 m,
BMI=60/(1.65)^2=22.04

Technologies Used:
Python 3.x

Installation:
Clone the repository
git clone https://github.com/yourusername/BMI-Calculator.git
Navigate to the project folder
cd BMI-Calculator

Run the program:
python bmi_calculator.py

 Example Output
Enter your weight in kilograms: 58
Enter your height in meters: 1.62

Your BMI is: 22.10
You are in the Normal weight category.
